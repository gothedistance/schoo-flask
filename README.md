Flask のアプリを起動する方法は下記の通りです。Windows の方は「python」だけで結構です。

```bat
python3 run.py
```

起動したら、下記のアドレスにアクセスして下さい。

http://127.0.0.1:5200

Flask の日本語のリファレンス一応あります。多少古いかも。
https://msiz07-flask-docs-ja.readthedocs.io/ja/latest/
