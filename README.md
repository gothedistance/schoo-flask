Flaskのアプリを起動する方法は下記の通りです。Windowsの方は「python」だけで結構です。

```bat
python3 run.py
```

起動したら、下記のアドレスにアクセスして下さい。

http://127.0.0.1:5000

Flaskの日本語のリファレンスも一応あります。多少古いかも。

https://flask-docs-ja.readthedocs.io/en/latest/
